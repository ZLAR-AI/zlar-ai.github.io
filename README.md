# zlar.ai

The public website for [ZLAR Inc.](https://zlar.ai) — OS-level governance for AI agents.

Hosted on GitHub Pages. Static HTML, no build step, no framework.

## Pages

| Page | Path | Description |
|------|------|-------------|
| Homepage | `index.html` | Product overview, all 7 ZLAR tools |
| ZLAR-OC | `zlar-oc.html` | OS-level containment for OpenClaw |
| ZLAR-CC | `zlar-cc.html` | Governance for Claude Code |
| ZLAR Gate | `zlar-gate.html` | Universal policy enforcement |
| ZLAR-LT | `zlar-lt.html` | Zero-config governance |
| ZLAR-AU | `zlar-au.html` | Audit trail analysis |
| ZLAR-NT | `zlar-nt.html` | Network egress policy |
| ZLAR-FL | `zlar-fl.html` | Fleet governance |
| Legal | `legal.html` | Terms, privacy, disclaimers |
| Bohm | `bohm.html` | Essay: "What the Audit Trail Sees" |
| Open Letter | `OPEN_LETTER.html` | Open letter to the industry |

## Installer

`install.sh` serves the ZLAR-LT zero-config installer via `curl -fsSL https://zlar.ai/install.sh | bash`.
